# Usage Examples

## 示例 1：完整翻译并输出带图 PDF

```text
调用 academic-pdf-translation-skill 翻译我上传的 PDF，并输出带原图的中文 PDF。
```

## 示例 2：只测试 PDF 提取

```text
调用 academic-pdf-translation-skill 处理我上传的 PDF。先只检查 PDF、提取文字、拆分 text blocks，不要翻译全文。
```

## 示例 3：只生成术语表

```text
调用 academic-pdf-translation-skill 处理我上传的 PDF。请只生成 paper_glossary.csv 和 merge_candidates.csv，不要翻译全文。
```
